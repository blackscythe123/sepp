PERSONAL FITNESS TRACKER

Practical Examination Question (Part C Implementation)

Scenario:

A fitness enthusiast wants a simple software system to track their daily workouts and monitor their Body Mass Index (BMI). The system should allow users to log exercises, view their history, and calculate their BMI using weight and height.

Requirements:

Functional Requirements:

  • Log workout with Exercise Name, Duration (in minutes), and Calories Burned (optional/calculated)
  • View all logged workouts in a table format
  • Calculate BMI using Weight (kg) and Height (m) inputs
  • Delete a workout record from the history
  • Basic data validation (ensure all fields are filled)
  • Maintain a log of all actions (log workout, calculate BMI, delete) with timestamps

Non-Functional Requirements:

  • Clean and readable layout
  • Immediate feedback for BMI calculations
  • Clear success and error messages
  • Confirmation before deleting any record

Implementation Guidelines:

  1. Build using only HTML and Vanilla JavaScript (No CSS allowed)
  2. Use in-memory arrays to store workout and log data
  3. Display results (Workouts, BMI, Log) in structured sections
  4. Include logic for BMI = Weight / (Height * Height)
  5. Ensure all JavaScript logic is contained within the HTML file

Deliverables:

Create a single HTML file (index.html) that fulfills all the requirements listed above. The application must be functional and demonstrate all required features clearly.

Marks Distribution (Total: 20 marks):

  • User Interface Structure: 4 marks
  • Workout Logging (Add/List): 5 marks
  • BMI Calculation Tool: 4 marks
  • Delete Functionality: 3 marks
  • Validation & Feedback: 2 marks
  • Activity Logging: 2 marks

Additional Challenge Questions (Recommended):

  • Add weekly workout summary with total duration and calories burned
  • Add intensity score calculation for each workout using duration and selected intensity level
  • Store BMI history and show trend compared to previous record

Time Limit: 60 minutes

Language: HTML + Vanilla JavaScript (No CSS)
