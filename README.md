# Management Systems Practice Repository

8 management systems implemented in HTML and Streamlit for exam preparation.

## How to Run

### HTML Implementation
Open any `index.html` file in a web browser:
```
HTML/01_StudentManagementSystem/index.html
HTML/02_LibraryManagementSystem/index.html
HTML/03_InventoryManagementSystem/index.html
HTML/04_HospitalManagementSystem/index.html
HTML/05_EmployeeManagementSystem/index.html
HTML/06_RestaurantManagementSystem/index.html
HTML/07_ProjectManagementSystem/index.html
HTML/08_EventManagementSystem/index.html
```

### Streamlit Implementation
Run any app.py file:
```bash
cd Streamlit/01_StudentManagementSystem
streamlit run app.py
```

Or any other system:
```bash
cd Streamlit/02_LibraryManagementSystem
streamlit run app.py
```
