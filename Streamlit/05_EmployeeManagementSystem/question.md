# EMPLOYEE MANAGEMENT SYSTEM - Exam Question

## Scenario
A company needs an HR management system to maintain employee records, manage departments, and generate reports. HR personnel should be able to add employees, update information, and generate analytics.

## Requirements

### Functional Requirements
- Add employee with ID, Name, Department, Position, Salary
- View all employees in directory
- Search employee by ID or Name
- Filter employees by department
- Update employee information
- Delete employee record
- Calculate total salary by department
- Generate department-wise employee report

### Non-Functional Requirements
- Operations within 30 seconds
- Support 50,000+ employees
- Real-time salary calculations
- Department filtering accuracy
- Employee count statistics

## Implementation Guidelines
1. Build using Python with Streamlit
2. Sections: Add Employee, View Directory, Search, Filter, Analytics
3. Store employee data using session state
4. Calculate department-wise salary totals
5. Generate employee count reports
6. Implement position-based hierarchy

## Deliverables
- Employee directory management system
- Employee database with all details
- Search and filter capabilities
- Department-wise analytics
- Salary statistics and totals
- Employee count per department
- Comprehensive reporting system

## Marks Distribution (Total: 20 marks)
- Employee Management: 3 marks
- Search & Filter: 4 marks
- Update Operations: 3 marks
- Salary Calculations: 3 marks
- Department Analytics: 4 marks
- Validation: 3 marks

---
**Time Limit:** 60 minutes
**Language:** Python (Streamlit)
